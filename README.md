# 『テトリスと代数学』

## 概要

本書は、代数学の基礎と、そのテトリスへの豊かな応用を体系的に学ぶためのオンライン教科書である。当リポジトリは、そのソースコードとコンテンツを管理するものである。全ての教材は、ウェブ上でどこからでも学習できるよう設計されている。

## 対象読者

本書は、テトリスの理論的側面に興味がある者に向けて執筆された。特に数学の知識を仮定しているわけではなく、論理的思考力と中学卒業程度の知識があれば読み進められる。

---

## 本書の特徴

- **詳細な証明と多数のRemark:** ほとんど全ての命題に詳細な証明をつけ、理解が難しいであろう箇所は直下にRemark(注意)を置くことによって、著者の考え方や定理の意味するところなどを学べるようにした。
- **豊富な演習問題:** 各セクションの末尾に、理解度を確認するための演習問題と簡潔な解答を配置している。
- **オープンソース:** 全てのコンテンツはオープンソースであり、誰でも自由に閲覧、改変、再配布が可能である。

## 貢献方法

本書への貢献を心より歓迎する。誤字脱字の報告、内容に関する提案、新たな演習問題の追加など、いかなる貢献も価値あるものである。

貢献にあたっては、以下の手順に従っていただきたい。

1.  当リポジトリをForkするか、Codespaceを作成する。
2.  変更内容に応じたブランチを作成する (`git checkout -b feature/your-feature`)。
3.  変更をコミットする (`git commit -m 'Commit message here'`)。
4.  作成したブランチをプッシュする (`git push origin feature/your-feature`)。
5.  プルリクエスト（Pull Request）を作成する。

貢献のガイドラインと詳細については、`CONTRIBUTING.md` を参照されたい。
